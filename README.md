# 🍹 SipVerse — Real-Time Blockchain Beverage Hub

SipVerse is a **real-time decentralized beverage ordering DApp** built on the Ethereum ecosystem.  
Users can browse premium beverages, add them to cart, pay securely with ETH through MetaMask, and leave permanent on-chain feedback stored immutably on the blockchain.

The project was initially **developed and tested locally using Hardhat**, then migrated to the **Ethereum Sepolia Testnet via Alchemy RPC**, with the frontend deployed live on **Vercel**.

---

## ✨ Features

- 🍸 **35+ beverages** across 7 categories  
  - Tops
  - Coffee
  - Tea
  - Drinks
  - Cocktails
  - Mocktails
  - Fruit Juice

- 🛒 Add-to-cart ordering flow
- 💳 ETH payments through MetaMask
- ⛓️ Permanent blockchain memos & feedback
- 🌐 Real-time Sepolia smart contract transactions
- 🌙 Luxury dark UI for premium user experience
- ☁️ Live frontend deployment on Vercel
